# 项目介绍

基于 electron 框架的云笔记应用.

## 完成功能：

1. markdown 编辑笔记
2. 文件保存到本地
3. 选择保存文件地址
4. 文件上传到七牛云
5. 导入本地 md 文件到应用
6. 自定义菜单
7. 自动版本更新
